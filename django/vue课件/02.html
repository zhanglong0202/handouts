2 Vue.js的概述

Vue.js读音 /vjuː/, 类似于 view

Vue.js是一个轻巧、高性能、可组件化的MVVM库，同时拥有非常容易上手的API

Vue.js是一个构建数据驱动的Web界面的库

Vue.js是一套构建用户界面的 渐进式框架

通俗的说:

  • Vue.js是一个构建数据驱动的 web 界面的渐进式框架

  • Vue.js 的目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件

  • 核心是一个响应的数据绑定系统
